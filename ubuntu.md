### - move folders and sub forders
```
$ sudo mv /fromPath/ /toPath/
```

### - check current directory
```
$ pwd
```

### - create directory
```
$ mkdir FILE_NAME
```

### - change directory
```
$ cd
```

### - check files/folders in directory
```
$ ls
```

### - check file/folder description
```
$ ls -l FILE_NAME
```

### - extract tar file
```
$ tar -xvf FILE_NAME
```
